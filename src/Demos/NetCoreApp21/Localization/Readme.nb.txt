Denne katalogen er ansvarlig for å levere alle
nødvendige filer som trengs for å lokaliseringstrenger
som kan/har blitt oversatt til andre språk.

Selv om filer som ender i `.en.resx` ikke trengs
generelt sett, å tilby disse filene kan gjøre det lettere
for oversettere som ønsker å oversette til andre spåk.
Avhengig av plattformen som brukes for lokalisering, kan det også
være nødvendig å være tilgjengelig for å laste opp de nødvendige
strengene til denne plattformen.

Denne katalogen brukes ikke som standard, men har blitt konfigurert
i metoden 'ConfigureServices' som er tilgjengelig i filen Startup.cs.
